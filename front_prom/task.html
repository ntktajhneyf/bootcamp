Задача:

Восстановить стили для страницы.
Представьте себе ситуацию, что это рабочий сайт, стили которого "потерялись". Вы не можете менять HTML, только писать CSS!
Надо сделать так, чтобы "все было как раньше".
По аналогичной "случайности" дизайн утерян, остались лишь скриншоты, хорошо что в png.
Скриншот дизайна (png ~921kb)
Ховеры дизайна (png ~880kb)
Используйте следующие медиа ресурсы:
Фон шапки (jpg ~1.4Mb)
Аватарка (jpg ~3.2kb)
Иконки не столь критичны, подберите любые похожие на свой вкус.
HTML дизайна

Критерии:

  • Точная реализация дизайна
  • Правильное "поведение" блоков при наведении
  • Максимально возможная кроссбраузерность с обоснованием
  • Graceful degradation

Результат:

Архив содерщащий:

  • html страничку с версткой (layout.html)
  • папка css с единственным файлом стилей (style.css)
  • папка img с необходимыми графическими материалами использованными в верстке
  • файл explain.txt c вашими комментариями/пояснениями к решению, обоснованиями кроссбраузерности и т.п.
